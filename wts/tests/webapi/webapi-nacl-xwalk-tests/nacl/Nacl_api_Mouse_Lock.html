Test Purpose:

Verifies the mouse movement is maintained even in tab and/or browser fullscreen

Test Steps:

  1. Execute command on IVI device with "app" user: "xwalk-launcher 'http://your_server:5103/api/mouse_lock/index.html?tc=pnacl'"
  2. Follow the opened index.html description to execute all steps
    • Move the mouse cursor over the grey box to watch the captured mouse movement, then move the mouse cursor to outside of grey box, the mouse movement unchanged;
    • Click on the grey box, the mouse cursor will be hidden;
    • Click on the grey box, then press the Enter key, the grey box will be fullscreen and the mouse cursor will be hidden;
    • Press the Esc key will cause two phenomenons, mouse cursor displayed and exited tab fullscreen mode, but fullscreen mode still valid;
    • Click on outside of the grey box, then press the F11 key will enter browser fullscreen mode, repeat execute step 1, press F11 key again will revoke browser fullscreen mode, but fullscreen mode still valid.

Expected Result:

  1. The command execute successfully
  2. The above steps execute successfully and the description correctly
